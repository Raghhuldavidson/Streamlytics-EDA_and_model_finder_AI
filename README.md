# Streamlytics 📊🤖

AI-Powered Data Analysis Workflow Inspired by Julius.ai

## Overview
Streamlytics is an AI-driven data analysis web application inspired by Julius.ai that automates exploratory data analysis (EDA), supports multi-format file handling, and generates intelligent insights using Large Language Models.

Built with Python and Streamlit, the platform enables users to upload datasets or documents, perform automated analysis, uncover patterns, and receive AI-powered insights through OpenAI and Gemini APIs.

## Features
- 📁 Multi-format file upload support
  - CSV
  - Excel
  - JSON
  - PDF

- 📊 Automated Exploratory Data Analysis (EDA)
  - Summary statistics
  - Missing value analysis
  - Correlation insights
  - Data distribution analysis

- 🤖 AI-powered analytical insights using:
  - OpenAI API
  - Gemini API

- 🧠 ML-powered intelligent recommendations
- 📈 Automated pattern detection and insight generation
- 💬 Natural language interaction for data understanding
- 🎯 Interactive Streamlit dashboard for seamless workflow

## Installation

### Clone Repository
```bash
git clone https://github.com/your-username/streamlytics.git
cd streamlytics
```

### Create Virtual Environment (Optional)
```bash
python -m venv venv
source venv/bin/activate      # Mac/Linux
venv\Scripts\activate         # Windows
```

### Install Dependencies
```bash
pip install -r requirements.txt
```

## Configure API Keys
Create a `.env` file:

```env
OPENAI_API_KEY=your_key_here
GEMINI_API_KEY=your_key_here
```

## Run Application
```bash
streamlit run streamlytics.py
```
